## 🔧 Troubleshooting & Common Issues

### ❓ The overlay menu does not appear after execution

Make sure you launched your execution environment with administrator privileges on Windows. On macOS, ensure accessibility permissions are granted in System Settings → Privacy & Security. Wait at least 10 seconds after injection before assuming failure.

### ❓ Features work but ESP elements flicker or disappear

This is typically caused by V-Sync conflicts. Disable V-Sync in both the popular title's graphics settings and your GPU driver control panel. Also try reducing `esp_draw_distance` in your configuration.

### ❓ The tool crashes immediately on Android

Verify you are running Android 9.0 or higher. Clear the execution environment's cache and data, then re-import the script file. If the issue persists, try downloading the latest release as older builds may have device-specific incompatibilities.

### ❓ Gun dupe process fails or produces errors

Switch `gun_dupe_mode` from `"fast"` to `"safe"` in your configuration file. The safe mode uses a more conservative timing approach that works across more server conditions. Ensure your network latency is stable — high ping can disrupt the timing sequence.

### ❓ Anti-detection warning appears during use

Immediately disable all active features using the panic key (default: `F12`). Wait 2–3 minutes before gradually re-enabling features one at a time. Update to the latest version if you are running an older release.

### ❓ macOS blocks the application from running

Navigate to System Settings → Privacy & Security and click "Allow Anyway" next to the blocked application notice. You may also need to run `xattr -cr` on the extracted folder from Terminal.

```bash
# Remove quarantine flag on macOS
xattr -cr ~/path/to/extracted/folder/
```
